# Sentinel-CDSS: Multi-Agent Clinical Triage System

**Live Demo:** https://sentinel-cdss-production.up.railway.app/
**GitHub:** https://github.com/ishantgupta30/sentinel-cdss

## What it does
AI-powered emergency room triage using deterministic NEWS2 scoring + multi-agent debate.

## Key Features
- Deterministic NEWS2 Engine (RCP 2017)
- Multi-Agent Debate: Clinical Expert + Logistics + Compliance + Audit Judge
- Antigravity: Parallel MCP tool calls
- Security Firewall: PHI scrubbing + injection detection
- Human-in-the-loop sign-off
- AUROC: 1.0000

## Tech Stack
- Google Gemini 2.5 Flash
- FastAPI + Streamlit
- SQLAlchemy + SQLite
- Deployed on Railway
